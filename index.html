HTML (Hypertext Markup Language) saytning kontenti uchun va
CSS (Cascading Style Sheets) saytning dizayni (ko'rinishi) uchun.
HTML va CSSni asosini urganish jarayonida avval siz oddiy restoran vebsaytini yaratasiz. Keyinchalik siz CSS o'zgaruvchilari kabi zamonaviy usullar yordamida pingvin yasaysiz. Shu bilan birgalikda bog'lana olishlik uchun eng yaxshi amaliyotlar yordamida veb formalari yaratasiz.
Va nihoyat, Flexbox yordamida Twitter karta va CSS Grid yordamida mukammal blog yasash orqali siz har xil ekran turlariga moslasha oladigan vebsaytlar yaratasiz.

Ushbu kursni tamomlab, siz ham shunday web sayt yarata olishingiz mumkin. HTML siznig dasturchi bo'lishingizda birinchi bosqich bo'lib, bilimingizga mustaxkam poydevor bo'ladi. Kirish

Demak, boshladik. Ishni oddiy vebsayt yaratishdan boshlaymiz. Buning uchun sizga editor kerak bo'ladi. Agar sizda Microsoft Windows o'rnatilgan bo'lsa, bloknot deb nomlangan matn editorini ochamiz. Editorga:
<h1>Salom</h1> deb yozamiz.
Bu yerda <h1> va </h1> HTML elementlari hisoblanadi. Ko'pchilik HTML elementlari ochuvchi va yopuvchi teglardan iborat bo'ladi.
Ochuvchi teg:
<h1>
Yopuvchi teg esa:
</h1>
Ko'rib turganingizdek, ochuvchi teg bilan yopuvchi teg orasidagi yagona farq bu ochuvchi burchakli qavsdan kyin turgan forward slash, yani oldinga enkaygan tayoqcha. /
"Salom" so'zini o'rnida o'zingiz istagan matnni yozishingiz mumkin. Endi ushbu faylni html formatida saqlaymiz.

h2 Elementi Yordamida Sarlavha Yozish

Ushbu darslarni urganish jarayonida uzingizni veb saytingizni bosqichma bosqich yaratib boring. h2 element sarlavxaning katta-kichikligi bo'yicha ikkinchi o'rnida turadi. h1 elementi asosan bosh sarlavha yozishda ishlatiladi, h2 elementi esa ikkilamchi sarlavha yozishda ishlatiladi. h1 va h2 dan tashqari, h3, h4, h5 va h6 elementlari ham mavjud.

Paragraflar

Katta hajmdagi matnni tartibga solishda paragraf juda muhim.
p elementi vebsaytta paragraf yasashda ishlatiladi. p harfi paragraf so'zining birinchi harfi hisoblanadi.
Paragraf quyidagicha yasaladi:
<p>Bu p teg!</p>
veb saytingizga h2 elementiga qo'shimcha p elementini ham qo'shib paragraf kiriting.
Esda tuting: Odat bo'yicha barcha html teglari kichkina harflar bilan yoziladi. Masalan:
<p></p> : to'g'ri yozilgan
<P></P> : noto'g'ri yozilgan

Matn Yozish

Ushbu kursni urganish jarayonida o'zingizning veb saytingizni yaratib boring. h1, h2, h3, h4, h5 va h6 elementlari yordamida sarlavha yozing. p tegi yordamida veb saytingizga matn yozing. Masalan: o'zingiz haqingizda qisqacha ma'lumot bering. Tug'ilgan yilingiz, sevimli mashgulotlaringiz va hokazolar.

HTML Kodni izox xolatidan chiqarish

Izox boshqa dasturchilarga eslatma qoldirish uchun foydalaniladi. Izoh veb saytning kod qismida yoziladi va vebsaytning o'zida kurinmaydi. Commenting is also a convenient way to make code inactive without having to delete it entirely. Izox kodning bir qismini vaqtinchalik uxlatib qo'yish uchun ham ishlatiladi. Vebsaytning uxlatilgan qismi veb saytga ta'sir qilmaydi. Izox HTMLda <-- bilan boshlanadi va --> bilan tugaydi

HTML kodni izox xolatiga ugirish

<-- ni kodning bir qismiga yoki kodning boshiga qo'yib --> ni kodning bir qismini tugash joyida yoki kodning eng so'ngiga qo'yib izox qanday natija berishini tekshirib ko'ring.

HTML Kodni o'chirish

h1 element bilan yozilgan tekst bazan telefonda ochilganda ekranga sig'masligi mumkin. Yasagan veb saytingizni telefonda muammosiz ochilishi uchun h1 urnida h2 ishlating.

HTML5 Elementlari Bilan Tanishuv

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. HTMLning birinchi versiyasi 1993 yilda Tim Berners-Lee tomonidan yozilgan. Undan buyon HTML5 introduces more descriptive HTML tags. These include main, header, footer, nav, video, article, section and others. These tags give a descriptive structure to your HTML, make your HTML easier to read, and help with Search Engine Optimization (SEO) and accessibility. The main HTML5 tag helps search engines and other developers find the main content of your page. Example usage, a main element with two child elements nested inside it: <main> <h1>Hello World</h1> <p>Hello Paragraph</p> </main> Note: Many of the new HTML5 tags and their benefits are covered in the Applied Accessibility section.
