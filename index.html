HTML dasturlash tili

Ushbu vebsayt faqat HTML dasturlash tili yordamida yaratilgan va sizga HTMLni bosqichma-bosqich urganishda qulay vosita bo'lib xizmat qiladi.

Ushbu kursni tamomlab siz ham shunday web sayt yaratishingiz mumkin. HTML siznig dasturchi bo'lishingizda birinchi bosqich bulib, bilimingizga mustaxkam poydevor bo'ladi. Dasturchilarga dunyo bo'ylab juda talab katta va dasturchilarning maoshi juda yuqori. Makrosoft kompaniyasi asoschisi Bil Geyts aytganidek: "dastulash qiyin emas". Odam qilgan ishni odam qila oladi.
